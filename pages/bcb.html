Base Curve Parts Sticking to B-side Troubleshooting

The following steps should be followed if you encounter parts sticking to the Base Curve B-side plates

Parts sticking to the BCB side mold plates is usually due to parts being overpacked.

Execute the following step in exclusively the prescribed order

1. Set the hold-pressure to the minimum limit as Per IOIDP-0122. As per IWA-0917

2. Set the V-P switch over to the maximum numerical value as per IOIDP-0122

3. Set the barrel temperature to the minimum limit

4. Set the back pressure to the minimum

5. Set the cooling time to the maximum limit if applicable as per IOIDP-0122. .

6. Increase the mold temperature to the maximum limit if applicable as per IOIDP-0122.

7. Check the suction cups for damage, verify the vacuum level

8. Swap vacuum lines on EOAT. If the location stays revert

9. Swap cup and stem on problem line, with known good location

10. Rotate plates. Does the problem stay or move? If still on the same cavity, get insert changed out. If it moves

A side plates/mold plates are suspect (Burn tips first)

11. Review /adjust the CTM/EOAT position as required for optimal alignment

8. Change out the plates
