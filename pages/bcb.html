Base Curve Parts Sticking to B-side Troubleshooting

The following steps should be followed if you encounter parts sticking to the Base Curve B-side plates

Parts sticking to the BCB side mold plates is usually due to parts being overpacked.

Execute the following step in exclusively the prescribed order

1. Set the hold-pressure to the minimum limit as Per IOIDP-0122.

2. Set the V-P switch over to the maximum numerical value as per IOIDP-0122

3. Set the cooling time to the minimum limit if applicable as per IOIDP-0122. .

4. Increase the mold temperature to the maximum limit if applicable as per IOIDP-0122.

5. Set the barrel temperature to the minimum limit if applicable as per IOIDP-0122.

6. Check the suction cups for damage, verify the vacuum level

7. Review /adjust the CTM/EOAT position as required for optimal alignment

8. Change out the plates
