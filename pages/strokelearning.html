Stroke learning Procedure

If you get resin build up / leakage between nozzle & sprue, you will need to bull back injection unit remove resin build up from the back of the sprue with a screwdriver.

When the sprue has been cleaned the injection unit will need to be retaught.

1. Close over mold fully in manual mode.

2. Using the jump function (the arrow key just to the left of the mouse), press arrow key then press C228 - this will bring you to page 1.4.

3. Set C228 from 0 to 1.

4. Select setup mode by turning key to 3 o clock position & pressing setup button(triangle & hand on button).

5. Press injection unit forward button & Green start button at the same time.

6. Unit will move forward & back for a couple of cycles. When injection unit teaching is completed C228 will go back to 0.

You can also access page 1.4 by clicking on topic menu - select mold movement the click on page - select 1.4 mold change - C228 will be available from here.

Or using short cut key S - keep pressing this button until page 1.4 comes up.

This procedure can also be accessed from the molding machine HMI,

From the overview screen click on function menu - select manual,

Then click on User manual - select setting/setup work - then select setting up mold.

Go to the bottom of this page & follow instruction for learning injection unit stroke.
