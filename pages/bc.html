Parts Sticking to A-side Troubleshooting

The following steps should be followed if you encounter parts sticking to the Base Curve A-side plates

Parts sticking to the BC A side mold plates is usually due to parts being underpacked.

Execute the following step in exclusively the prescribed order

1. Set the hold-pressure to the maximum limit as Per IOIDP-0122.

2. Set the V-P switch over to the minimum numerical value as per IOIDP-0122

3. Set the first mold opening position to at least 2 mm and Increase the first stage mold opening speed to 70 ~95 %

4. Set the cooling time to the minimum limit if applicable as per IOIDP-0122.

5. Set the mold temperature to the minimum limit if applicable as per IOIDP-0122.

6. Set the barrel temperature to the maximum limit if applicable as per IOIDP-0122.

7. Burn-off the tips

8. Change out the plates
